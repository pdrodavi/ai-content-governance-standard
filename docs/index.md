# Padrão Público de Governança de Conteúdo para IA Generativa

Este site publica a versão navegável do **AI Content Governance Standard**.

O foco do padrão é tratar **governança como problema de engenharia**, com:
- policy-as-code
- separação entre decisão normativa e LLM
- auditoria e rastreabilidade ex post
- alinhamento regulatório multijurisdicional

Use o menu à esquerda para navegar pelos capítulos.
